Termos técnicos

Bug - Erro, falha ou defeito em algum componente envolvendo código-fonte.
Código-fonte - Instruções em uma linguagem de programação para execução de um programa ou aplicação.
Increment - Conceito utilizado na metodologia Scrum para quantificar o progresso que uma certa tarefa adiciona ao produto final.
Minimum Viable Product - (tradução: Mínimo Produto Viável). Conceito utilizado na metodologia Scrum para quantificar o estado mínimo para um produto ser considerável viável para entrega, normalmente dependente de tarefas críticas.
Story Point - Pontuação usada para descrever a dificuldade/esforço necessário para compleção de uma tarefa, utilizada internamente na formação do Product Backlog.
HTML - Linguagem de marcação de arquivos de páginas da Web.
CSS - Linguagem para descrição da apresentação de um documento escrito em HTML ou em XML.
Jira - Plataforma de gerenciamento de projetos.
